# 🧠 General Knowledge Trivia Game

A fun and interactive web-based trivia game that tests your general knowledge across various topics including geography, science, history, art, literature, and mythology.

## 🎮 Game Features

- **10 Challenging Questions**: Covering diverse topics for a comprehensive knowledge test
- **Interactive UI**: Clean, modern design with smooth animations and transitions
- **Real-time Feedback**: Immediate feedback showing correct/incorrect answers
- **Score Tracking**: Keep track of your performance throughout the game
- **Progress Indicator**: Visual progress bar showing game completion
- **Results Screen**: Detailed final score with performance evaluation
- **Answer Review**: Review all questions and see where you went right or wrong
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Accessibility**: Screen reader friendly with proper ARIA labels

## 🚀 How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin the trivia challenge
3. Read each question carefully and select your answer
4. Get immediate feedback on whether you're correct or not
5. Progress through all 10 questions
6. View your final score and performance evaluation
7. Review your answers to learn from mistakes
8. Click "Play Again" to test your knowledge again!

## 🌐 Running the Game

### Option 1: Direct Browser Access
Simply open `index.html` in your web browser.

### Option 2: Local Server (Recommended)
For the best experience, serve the files through a local web server:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Python 2
python -m SimpleHTTPServer 8080

# Using Node.js (if you have live-server installed)
npx live-server

# Using PHP
php -S localhost:8080
```

Then visit `http://localhost:8080` in your browser.
